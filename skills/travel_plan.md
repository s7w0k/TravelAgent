# Travel Plan Skill

旅行计划规划技能，负责协调 12306 票务查询、天气查询和高德地图导航，为用户提供完整的旅行计划服务。

## 职责

- 接收用户的旅行需求（出发地、目的地、时间等）
- 调用 MCP 工具查询实时信息
- 执行行程验证，确保计划合理性
- 整合信息生成完整的旅行计划

## 工作流程

1. **需求分析**：解析用户的旅行需求，提取关键信息
2. **信息查询**：按需调用 MCP 工具获取实时数据
3. **行程验证**：检查计划合理性
4. **计划输出**：使用标准格式输出旅行计划

## 输出格式

```markdown
## 📋 旅行计划

### 🚄 交通信息
- 车次：[车次号]
- 出发时间：[时间]
- 到达时间：[时间]
- 余票情况：[票种及数量]

### 🌤️ 天气预报
- 日期：[日期]
- 天气：[晴/雨/多云]
- 温度：[最低~最高℃]

### 🗺️ 目的地导航
- 从车站到目的地：[路线描述]
- 预计耗时：[时间]

### ✅ 行程验证
- 到站时间：[合理/需注意]
- 天气状况：[适宜/需注意]

### 📅 推荐行程
[详细行程安排]

### 💰 预算估算
- 总计：¥[金额]
```

## 策略说明

### 备选方案策略
- 首选车次无票时，推荐相邻车次
- 直达无票时，推荐中转方案

### 行程验证规则
- 避免深夜 23:00-6:00 到达
- 天气预警时调整户外活动
