# 12306 MCP Skill

12306 火车票余票查询技能，通过 12306 MCP Server 接入中国铁路官方票务系统，支持全车型查询。

## MCP 配置

```json
{
  "mcpServers": {
    "12306-mcp": {
      "command": "npx",
      "args": ["-y", "12306-mcp"],
      "transport": "stdio"
    }
  }
}
```

## 环境要求

- Node.js 18+
- npx 可用

## 功能

- 余票查询：支持 G/D/C/Z/T/K/普快 所有车型
- 车站搜索：支持中文/拼音/简拼
- 中转换乘：查询一次中转的换乘方案
- 经停站查询：查询列车经停站及时刻表

## 注意事项

- 查询时间建议在 6:00-23:00 之间（12306 服务时间）
- 预售期为 15 天
- 节假日期间余票变化较快，建议实时查询
- 本项目仅供学习研究，请遵守 12306 官方相关规定
